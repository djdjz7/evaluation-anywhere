## 浏览器兼容性

> [!NOTE]
> 开发环境：  
> Microsoft Edge 121+  
> Apple Safari on iOS 17

为与新测评行为对齐，图片的上传格式应为 WEBP。在 Google Chrome
Microsoft Edge 及 Mozilla Firefox 中，画板及上传的图片能够转换为
WEBP；然而， Safari 并不支持 Canvas 到 WEBP 的转换，故 Safari
上使用 PNG 格式上传。
